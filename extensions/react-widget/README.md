# DocsGPT react widget


THis widget will allow you to embed a DocsGPT assistant in your react app.

## Installation

```bash
npm install  docsgpt
```

## Usage

```javascript
    import { DocsGPTWidget } from "docsgpt";
    import "docsgpt/dist/style.css";

    const App = () => {
      return <DocsGPTWidget />;
    };
```
